# Web 1 Final Project

A complete, responsive temple museum website built for Web 1 coursework.

## Tech Stack
- HTML5 (semantic structure)
- CSS (Flexbox, Grid, media queries)
- JavaScript (UI interactivity)

## Main Features
- One-page smooth scrolling layout
- Filterable temples collection with modal details
- Responsive design for desktop and mobile
- Visit planner form with validation and success feedback
- Media integration (images + map iframe)
- Pass options table

## Run Locally
Open index.html in your browser.
